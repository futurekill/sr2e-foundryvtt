# Creature Portrait Prompts

Image-generation prompts for the SR2E critters compendium, in the same art
style as the four hand-made samples (`banshee`, `barghest`, `cockatrice`,
`ghoul` in `assets/creature_portraits/`).

**How to use:** paste **`SUBJECT` + the `STYLE SUFFIX`** into ChatGPT image
generation (or any image model). The suffix is what keeps every portrait
consistent with the samples — don't edit it between creatures. Generate at a
**3:4 portrait** aspect ratio (e.g. 1024×1365). Save each as
`assets/creature_portraits/<name>.webp` using the filename noted per entry, and
I'll wire it into the actor the same way I did the first four.

> Neither I nor the Codex CLI can generate images — these are text/code tools.
> These prompts are the deliverable; the rendering happens in your image model.
